hexo

hexo是一个快速、简单和强大的博客框架，可以帮助我们快速搭建一个静态博客并部署到github上面，具体使用方法如下：

一、初始化博客

  1. 安装node.js

    在node官网下载长期支持版(LTS)进行安装，会安装上node.js和npm包管理器

    检查是否安装命令：node -v npm -v

    切换成淘宝镜像可以加快下载速度，参考：菜鸟教程

  2. 全局安装hexo-cli

    通过hexo -v 验证是否安装了hexo

  3. 创建存放博客的文件夹以及初始化博客

  4. 切换到目标文件夹cd blog

  5. 安装包依赖npm install

  6. 在本地启动博客项目hexo server

三、更换博客主题

这里以Meadow主题为例

  1. 选择喜欢的主题 地址：https://hexo.io/themes/

  2. 克隆主题到本地(参考具体主题的克隆地址)

  3. 修改根目录下面的_config.yml文件中的themes

  4. 在本地查看效果

四、将博客部署到github上面

  1. 安装hexo-deployer-git插件

  2. 在github上创建仓库并配置

    创建的仓库名称为：mapcode001.github.io
