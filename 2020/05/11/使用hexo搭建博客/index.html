hexo

hexo是一个快速、简单和强大的博客框架，可以帮助我们快速搭建一个静态博客并部署到github上面，具体使用方法如下：

一、初始化博客

  1. 安装node.js

    在node官网下载长期支持版(LTS)进行安装，会安装上node.js和npm包管理器

    检查是否安装命令：node -v npm -v

    切换成淘宝镜像可以加快下载速度，参考：菜鸟教程

  2. 全局安装hexo-cli

    通过hexo -v 验证是否安装了hexo

  3. 创建存放博客的文件夹以及初始化博客

  4. 切换到目标文件夹cd blog

  5. 安装包依赖npm install

  6. 在本地启动博客项目hexo server
